Angular Job Dashboard
Simple Job Dashboard application, that is easy to understand and extend. Developed by using Anna agent.

N|Solid

The application currently supports Angular 17.0.0 version
Features
A simple Job Dashboard application, allowing to list all job available that matches your resume.

Installation
Angular Job Dashboard requires Node.js latest to run, but you can try your luck with previous versions )).

Install the dependencies and devDependencies and start the server.

cd angular-todo
npm i
Development
RUN
To run the application :

npm start

Note: Application uses 4200 port as default. Want to change the port? Then go to package.json and change ng serve --open to ng serve --port XXXX --open

BUILD
To build the application run:

ng build

UNIT TESTS
To execute the unit tests run:

ng test

END 2 END UNIT TESTS
To execute the end-to-end unit tests run:

ng e2e
