# Angular Job Dashboard
## Features
- A simple Job Dashboard application, allowing to list all job listing that matches your uploaded resume.

## Installation

Angular Job Dashboard requires [Node.js](https://nodejs.org/) latest to run, _but you can try your luck with previous versions_ )).

Install the dependencies and devDependencies and start the server.

```sh
cd angular-todo
npm i
```

## Development
### RUN
run the main application:
```sh
npm start
```
> Note: Application uses 4200 port as default. 
Want to change the port? 
Then go to `package.json` and change `ng serve --open` to `ng serve --port XXXX --open`

### BUILD
To build the application run:
```sh
ng build
```

### UNIT TESTS
To execute the unit tests run:
```sh
ng test
```

### END 2 END UNIT TESTS
To execute the end-to-end unit tests run:
```sh
ng e2e
```
